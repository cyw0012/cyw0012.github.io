# 软件分享

首先我先介绍一下许多办公类的软件，如何选择下载？

这里介绍两个：

  1. 网站：小熊软件助手 (opens new window)

  2. 公众号：搜索并关注“火耳软件管家”公众号，在聊天界面打开软件目录，选择自己需要的看教程下载。

我推荐这两个的原因：免费、有详细的安装教程。

我推荐一些软件：

百度网盘链接：https://pan.baidu.com/s/1jBIS5U4j2AGuuTtpTjQZzQ?pwd=21yl

提取码：21yl

  1. Listen1：免费听各大平台的音乐，还支持登录功能（可查看自己的收藏、喜欢的音乐）。

  2. Typora：非常简洁的界面，可沉浸式的做笔记，语法非常简单。

    🦀 学习语法：基本语法 (opens new window) 高级技巧 (opens new window) 更多数学符合表达式 (opens new window) LaTeX 表示方法 (opens new window) Emoji (opens new window)

  3. XMind：思维导图，让结构变得清晰明了。

  4. IDM、Motrix：可加速下载资源。

  5. geek：彻底、干净卸载软件。

  6. Snipaste：强大的截图软件，可自动检测窗口、可贴图。

  7. uTools：实用工具，里面有非常多的插件。

  8. He3：超多开发工具。

  9. github加速器：加速打开Github网站。

  10. Git：分布式版本控制系统，下载网址 (opens new window)。
