# 二、SEO 术语

方便准确的在浏览器搜索到自己需要的内容。

  1. 完全匹配："关键字"

  2. 排除：关键字 -需排除的关键字

  3. 增加：关键字 +需增加的关键字

  4. 指定域名搜索：关键字 site:域名（如：怎么学好英语 site:zhihu.com）

  5. 指定文件搜索：关键字 filetype:文件类型

  6. 通配符符搜索（模糊搜索）：* 用来代替你不知道的字

  7. 指定搜索结果中要包含的关键字：关键字 intitle:关键字

  8. 限定时间范围搜索：关键字 时间..时间

# 三、常用命令行

博客 (opens new window)

  1. 上箭头：回到上一命令

  2. Tab 键：快速补全路径

  3. 进入盘符

    d:

  4. 操作文件

    # 创建文件夹
md 文件夹名

# 删除文件夹
rd 文件夹名

# 删除文件
del 文件名(*.txt删除文件类型为txt的全部文件)

  5. 改变当前目录

    # ①用绝对路径切换
cd d:\day01

# ②切换到当前目录的day01目录下
cd day01

# ③切换到上一级目录
cd…

# ④切换到盘符根目录
cd/

  6. 清空任务板

    cls

  7. 退出

    exit

  8. 查看当前文件夹内容

    dir
