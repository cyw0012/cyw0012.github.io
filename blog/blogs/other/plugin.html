# 插件与脚本

这个可以让你的浏览器变得非常简洁，然后能去除广告，免费听歌、看电视剧、网盘加速下载、随便复制等等。我不允许还有人不知道！！！

获取浏览器商店的插件：

打开浏览器->点击右上角（有三个点）->（打开设置）选择扩展->打开开发人员模式->打开浏览器商店获取扩展。

获取更多的插件：

Crx搜搜 (opens new window)，这包含了许多浏览器的插件，应有尽有。下载完后把crx文件拖拽到浏览器的扩展界面即可。

# 一、插件

  1. Tampermonkey：油猴，用户脚本管理工具（这个必下）。

  2. Listen 1：各大平台的音乐，还可支持登录获取自己的收藏喜欢的音乐。

  3. AIX智能下载器(图片/视频/音乐/文档)：可获取网页的图片、音乐、视频资源。

  4. WeTab-免费ChatGPT新标签页：强烈推荐，让你的新标签页变得简洁、美观。

  5. 轻氧翻译：可在网页上选中文字直接翻译。

  6. 电子书全网搜：全网搜索电子书。

  7. IDM Integration Module：配合IDM进行高速下载。

  8. Simple Allow Copy：网页文字随便复制。

# 二、油猴脚本

在下载好Tampermonkey插件后，在任务栏中点击油猴图标，选择“获取新脚本”，然后建议在GreasyFork中获取新脚本。

  1. AC-baidu-重定向优化百度搜狗谷歌必应搜索_favicon_双列：优化搜索界面，在搜索界面有控制面板可自行设置。

  2. 全网视频VIP多接口免费解析观看：所有平台免费看电视剧。

  3. 【屏蔽广告】屏蔽谷歌广告、百度广告、知乎广告、隐藏谷歌和百度搜索增强百度搜索结果的各种广告等等（过滤所有采用谷歌联盟和百度联盟等广告联盟的广告）。

  4. 网盘智能识别助手：选中网盘网站和密码，可自动打开。

  5. 网页加速器。

  6. 夜间模式助手。

  7. 网盘直链下载助手：可配合Motrix、IDM软件使用。

  8. B站哔哩哔哩使用增强，全网VIP视频免费破解去广告，全网音乐直接下载，知乎使用增强，短视频无水印下载，油管、Facebook等国外视频解析下载，网盘搜索引擎破解无限下载等。

  9. Github 增强 - 高速下载。

  10. GitHub汉化插件。

  11. 文本选中复制。
