# jQuery

# 一、jQuery 入门

  1. JavaScript 库：即 library，封装好的集合。(就是 js 文件，对原生 js 进行封装）

    常见的 JS 库：jQuery、Prototype、YUI、Dojo、Ext JS、移动端的 zepto。

  2. jQuery 就是快速、简洁的 js 库，可快捷操作 DOM，里面基本是函数（方法）。

  3. 优点：轻量级、跨浏览器兼容、链式编程、隐式迭代、简化了 DOM 操作、支持插件扩展开发、免费开源。

  4. jQuery 的使用：jQuery 官网 (opens new window)

    下载页面中第一个是压缩过的；第二个未压缩。

  5. jQuer 入口函数：相当于 DOMCOntentLoaded

    $(function() {
	...
});

  6. jQuery 的顶级对象 $

    • $ 是 jQuery 的别称

    • $ 是 jQuery 的顶级对象

  7. DOM 对象：就是用原生 js 获取到的对象；

    jQuery 对象：就是用 jQuery 方式获取到的对象；（是伪数组）。

    DOM 对象只能用原生 js 的属性和方法，jQuery 对象只能用 jQuery 方法。

  8. jQuery 对象和 DOM 对象相互转换：

    // 1.DOM对象转jQuery对象
let myvideo = document.querySelector('video')
$('video')
$(myvideo)
// 2.jQuery对象转DOM对象
$('video')[0].play()
$('video').get(0).play()

# (1) jQuery 选择器

$("选择器")

  1. 基本选择器：

    名称 用法 描述
    ID 选择器 $("#id") 获取指定 ID 的元素
    全部选择器 $("*") 匹配所有元素
    类选择器 $(".class") 获取同一类 class 的元素
    标签选择器 $("div") 获取同一类标签的所有元素
    并集选择器 $("div,p,li") 选取多个元素
    交集选择器 $("li.current") 交集元素
    子代选择器 $("ul>li") 获取亲儿子层级元素，不能获取孙子
    后代选择器 $("ul li") 获取后代的所有元素

    TIP

    隐式迭代：把匹配的所有元素内部进行遍历循环，然后进行操作（对同一类元素做同样操 作）。

  2. 筛选选择器：

    //获取第一个、最后一个元素:first、:last
$('ul li:first').css('background-color', 'red')
// 获取指定索引号（从0开始）元素:eq(index)
$('ul li:eq(2)').css('background-color', 'blue')
// 选中索引号为奇数的元素
$('ol li:odd').css('background-color', 'pink')
// 选中索引号为偶数的元素
$('ol li:even').css('background-color', 'green')

    $(this)是 jQuery 当前对象

  3. 其他选择器：

    语法 用法 说 明
    parent() $("li").parent(); 查找父 级
    parents() $("li").parents(); 找到所有祖宗元 素
    children(selector) $("ul").children("li") 相当于$ ("u1>1i"),最近一级（亲儿子）
    find(selector) $("ul").find("li"); 相当于$("ui"), 后代选择器
    siblings(selector) $(".first").siblings("li"); 查找兄弟节点， 不包括自己本身
    nextAll([expr]) $(".first").nextAll() 查找当前元素之 后所有的同辈元素
    prevtAll([expr]) $(".last").prevAll() 查找当前元素之 前所有的同辈元素
    hasClass(class) $("div").hasclass("protected") 检查当前的元素 是否含有某个特定的类，如果有，则返回true
    eq(index) $("li").eq(2); 相当于$("li:eq (2)"),index 从 0 开始

TIP

链式编程：简洁

$(this).css('color', 'red').siblings().css('color', '');

# (2) jQuery 样式操作

  1. 操作 css：

    • 参数只写属性名，则返回属性

    • 参数有属性名和值，则修改属性值（数字可不加引号）

    • 参数为对象，则可设置多组样式（如果是复合属性则要驼峰命名法，值不是数字要加引号）

  2. 设置样式

    • 添加类
    $('div').addClass('current')
    • 删除类
    $('div').removeClass('current')
    • 切换类
    $('div').toggleClass('current')

    注：原生 js 的 className 会覆盖原来的，而 jQuery 不会。

# (3) jQuery 效果

  1. 显示隐藏

    • 显示：show([speed],[easing],[fn])

    • 隐藏：hide([speed],[easing],[fn])

    • 切换：toggle([speed],[easing],[fn])

    speed：表示速度，为 slow、normal、fast，或者为毫秒值；easing：指定切换效果，为 swing、linear；fn：回调函数，动画完成后执行。

  2. 滑动

    • 下拉：slideDown([s],[e],[fn])

    • 上拉：slideUp([s,[e],[fn])

    • 滑动切换：slideToggle([s],[e],[fn])

  3. 淡入淡出

    • 淡入：fadeIn([s],[e],[fn]) 说明 (opens new window)

    • 淡出：fadeOut([s],[e],[fn]) 说明 (opens new window)

    • 修改透明度：fadeTo([s],[e],[fn]) 说明 (opens new window)

    • 淡入淡出切换：fadeToggle([s],[e],[fn]) 说明 (opens new window)

  4. 自定义动画

    • animate()

补充：hover 是鼠标经过、离开事件

stop()可阻止动画排队，必须写在动画前面

# (4) jQuery 属性操作

  1. 获取固有属性：element.prop("属性名")

  2. 设置固有属性：element.prop("属性名","属性值")

  3. 获取自定义属性：element.attr("属性名")

  4. 设置自定义属性：element.attr("属性名","属性值")

  5. 数据缓存（可在指定元素上存取数据，不修改 DOM 结构，刷新则释放）：data("name","value")

  6. 获取数据：data("name") 也可以获取 h5 新增的 第一个、没有 data-、而且返回的是数字型

# (5) jQuery 内容文本值

  1. 普通元素内容：html() html("修改内容")

  2. 元素文本内容：text() text("修改内容")

  3. 表单值：val() val("修改内容")

# (6) jQuery 元素操作

  1. 遍历元素

    展开代码 
    // 1.each()方法遍历元素
$('div').each(function (i, domEle) {
//回调函数的两个参数：一个是索引号，一个是DOM对象（可自己命名）
    ...
})
// 2.$.each()主要用于遍历数据
// 遍历元素
$.each($('div'), function (i, domEle) {
    $(domEle).css('color', arr[i]);
})
// 遍历数组
$.each(arr, function (i, ele) {
    console.log(i); //索引
    console.log(ele); //值
})
// 遍历对象
$.each({
    name: "cyw",
    age: 19
}, function (i, ele) {
    console.log(i); //属性
    console.log(ele); //值
})

  2. 创建元素

    let li = $('<li>li2</li>')

  3. 添加元素

    // 内部添加
$('ul').append(li) //(放到后面)
$('ul').prepend(li) //(放到前面)
// 外部添加
let div = $('<div>div</div>')
$('ul').after(div) //(放到后面)
$('ul').before(div) //(放到前面)

  4. 删除元素

    // 删除自己
$('div').remove()
// 删除所有子节点
$('ul').empty()
$('ul').html('')

# (7) jQuery 尺寸、位置操作

  1. 尺寸

    语法 用法
    width()/height() 取得匹配元素宽度和高度值只算 width/height
    innerWidth()/innerHieght() 取得匹配元素宽度和高度值包含 padding
    outerWidth()/outerHeight() 取得匹配元素宽度和高度值包含 padding、border
    outerWidth(true)/outerHeight(true) 取得匹配元素宽度和高度值包含 padding、borde、margin

  2. 位置

    • offset()：设置或获取距离文档的位置（传对象可修改位置）

    • position()：获取带有定位父级位置（不可以设置）

    • scrollTop()、scrollLeft()：设置或获取被卷去的头部和左侧

# (8) jQuery 事件

  1. 事件注册：

    element.事件(function () {}) //单个事件注册

  2. 事件处理：

    • on()可绑定一个或多个事件、也可以实现事件委托、可以实现给未来动态创建的元素绑定事件。

      element.on(events, [selector], fn) //事件类型，子元素选择器，函数

    • 触发事件一次：one()

    • 自动触发事件：

      // 1.简写
$('div').click()
// 2.trigger()
$('div').trigger('click')
// 3.triggerHandler() 不会触发元素的默认行为
$('div').triggerHandler('click')

  3. 事件解绑：off()

    $('div').off() //解除div上的所有事件
$('div').off('mouseover') //解除div上的mouseover事件
$('ul').off('click', 'li') //解除ul的事件委托

# (9) jQuery 拷贝对象

  1. 浅拷贝：把原来对象里面的复杂数据类型地址拷贝给目标对象（修改会影响其值）

    $.extend(target, obj) //会覆盖原来的数据

  2. 深拷贝：把里面的数据完全复制给目标对象，如果有不冲突的属性，则会合并到一起

    $.extend(true, target, obj)

# (2) 电梯导航案例

要点：

  1. 卷到某处就显示或隐藏

  2. 绑定点击事件，页面滚动到相应位置（获取索引号，拿到位置距离）

  3. 电梯导航状态发生相应变化

  4. 滚动到相应位置，电梯导航发生相应变化

  5. 设置节流阀（互斥锁），防止在点击电梯导航时触发滚动事件

注：带动画的返回顶部：是元素做动画，因此$("body,html")。

# (3) toDoList 案例

  1. 核心思路：敲回车、点击删除按钮、点击复选框，都是把本地存储的数据加载到页面中（页面刷新不丢失数据）

    具体实现：获取->操作->存储(更新)->渲染

  2. 存储格式为：数组对象 let todolist=[{title: 'xxx', done: false}]

  3. 数组对象转为字符串：JSON.stringify(data)

  4. 字符串转为数组对象：JSON.parse()
