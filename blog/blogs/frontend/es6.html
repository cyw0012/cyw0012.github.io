# ES6

# 一、面向对象

  1. 面向对象介绍：

    • 面向过程(POP)：按照分析好的步骤解决问题；性能高。

    • 面向对象(OOP)：按照对象的功能解决问题；易维护、易复用、易扩展。

    面向对象的三个特性：

    • 封装性

    • 继承性

    • 多态性

  2. ES6 中的类和对象

    类是抽象的，对象是具体的（是类的实例化）。

    类抽象了对象的公共部分。

    对象是由属性和方法组成。

    // 1.创建类
class Star {
  constructor(uname, age) {
    //构造函数(创建对象时自动调用)
    this.uname = uname
    this.age = age
  }
}

// 2.创建对象
let ldh = new Star('刘德华', 23)
console.log(ldh)
// 注：
// 1.类里面的所有函数不要写function
// 2.多个函数不要加逗号

  3. 类的继承

    子类继承父类的一些属性和方法

    class Father {}
class Son extends Father {} //子继承父

    • super() 可调用父类的构造方法（super 必须在子类 this 之前调用）

    • super.XXX() 调用父类的普通函数

    • 在 ES6 中没有变量提升，先定义类再实例化

    • 使用类里面的共有属性和方法一定要加 this

    • constructor 中的 this 指向的是 创建的实例对象；方法中的 this 指向调用者

    • insertAdjacentHTML('beforeend', li)可以向父元素添加字符串


# 二、原型

# (1) 构造函数和原型

  1. 构造函数：首字母要大写；要 new 一下

    实例成员：通过 this 添加的，只能通过实例化的对象来访问

    静态成员：在构造函数本身添加成员，只能通过构造函数访问

    存在内存浪费

  2. 原型对象(prototype)：所有对象共享，所有公共属性、方法放到原型上。

    对象原型(Prototype、_ proto_)：对象上系统自动添加一个原型，指向构造函数的原型对象。

    构造函数的原型对象有一个属性 constructor(构造函数本身)。

  3. 原型链：

    只要是对象就有对象原型；而原型对象的对象原型(_ proto_)指向的是 Object.prototype；Object.prototype 的原型对象指向为 null

    • 原型链查找规则：一级一级往上找，找不到为 undefined。

    • 原型对象 this 指向：原型对象的 this 也指向实例对象

    • 扩展内置对象

      Array.prototype.sum = function () {}

# (2) 继承：

  • call()：调用函数，并修改函数运行时的 this 指向

    fun.call(thisArg,arg1,arg2...)
//thisArg为调用函数的this指向
//arg1,arg2...为形参

    // ES6之前
// 父构造函数
function Father(uname, age) {
  this.uname = uname
  this.age = age
}
// 子构造函数
function Son(uname, age, score) {
  // 让父构造函数的this指向子构造函数的this
  Father.call(this, uname, age)
  this.score = score
}
// Son的原型对象指向Father的实例对象，从而继承到原型对象
Son.prototype = new Father()
SOn.prototype.constructor = Son

let ldh = new Son('刘德华', 18, 100)
console.log(ldh)

  • 类的本质：

    类的本质就是一个函数，构造函数的另一种写法；类有原型对象；类的原型对象的 constructor 指向本身；类可以通过原型对象添加方法；类的实例对象也有对象。

    ES6 中的类就是语法糖（便捷写法）


# 三、ES6 新增方法

  1. 迭代方法：forEach()、map()、filter()、some()、every()

    let arr = [12, 23, 45, 3]
// 参数用不到可不写
// forEach的使用
arr.forEach(function (value, index, array) {
  console.log('每个数组元素的值：' + value)
  console.log('每个数组元素的索引：' + index)
  console.log('数组本身：' + array)
})
// filter的使用(筛选数组) 返回满足条件的新数组,不影响原数组
let newArr = arr.filter(function (value, index) {
  return value % 2 === 1
})
console.log(newArr)
// some方法 返回一个布尔值(如果找到则立即退出)
// 遇到return，forEach、filter不能终止迭代，some能终止迭代，效率高（这里必须为true，否则会继续迭代）

  2. 定义、修改对象属性：Object.defineProperty(obj,prop,descriptor)

    参数为：目标对象、属性名、目标属性特性

    • value:设置属性的值默认为 undefined

    • writable::值是否可以重写。true|false 默认为 false

    • enumerable:目标属性是否可以被枚举。true|false 默认为 false

    • configurable:目标属性是否可以被删除或是否可以再次修改特性 true|false 默认为 false

    Object.keys()：遍历对象，返回由对象属性组成的数组

    delete 对象.属性：可删除属性


# 四、函数进阶

函数执行完变量销愧。

  1. 函数定义和调用方式

    所有对象都是 Function 的实例(对象)；函数也属于对象

  2. this 指向

    调用方式 this 指向
    普通函数调用 window
    构造函数调用 实例对象 原型对象里面的方法也指向实例对象
    对象方法调用 该方法所属对象
    事件绑定方法 绑定事件对象
    定时器函数 window
    立即执行函数 window

    改变函数 this 指向：call()、bind()、apple()

    // 1.call() 主要作用是实现继承
// 2.apply() 它的参数必须是数组(伪数组)
// 主要作用：借助数学对象求最大值、最小值
// apply()精髓就是把传入的数组参数打碎变成参数列表，然后把这个参数列表再传递给原来的参数
// 3.bind() 不调用函数,返回的是改变this之后产生的新函数

  3. 严格模式：IE10 以上支持

    • 消除了 Javascript 语法的一些不合理、不严谨之处，减少了一些怪异行为。

    • 消除代码运行的一些不安全之处，保证代码运行的安全。

    • 提高编译器效率，增加运行速度。

    • 禁用了在 ECMAScript 的未来版本中可能会定义的一些语法，为未来新版本的 Javascript 做好铺垫。比 如一些保留字如：class,enum,export,extends,.import,super 不能做变量名

    为脚本开启严格模式：严格模式应用到整个脚本

    为函数开启严格模式：严格模式应用到个别函数

    'use strict';

    // 1.变量需先声明在使用
// 2.不能随意删除已经声明的变量
// 3.在全局作用域中的函数this为undefined
// 5.定时器的this还是指向window
// 6.事件、对象还是指向调用者
// 7.不允许重名

  4. 高阶函数：接收函数作为参数或将函数作为返回值输出（函数也是一种数据类型）

  5. 闭包：一个作用域访问另一个函数作用域的内部变量的函数

闭包的作用：延伸了变量的作用范围（执行完了变量不销愧，等别的函数调用完了再销愧）

  1. 递归：如果一个函数在内部可以自己调用自己（容易栈溢出）

  2. 浅拷贝：拷贝简单数据类型，拷贝的有复杂数据类型时，拷贝的是地址。

    // es6方法 浅拷贝
Object.assign(o, obj)

  3. 深拷贝：拷贝所有数据

    function deepCopy(newobj, oldobj) {
  for (let k in oldobj) {
    let item = oldobj[k]
    // 1.判断这个值是否为数组(放在前面，因为数组也是对象)
    if (item instanceof Array) {
      newobj[k] = []
      deepCopy(newobj[k], item)
    }
    // 2.判断这个值是否为对象
    else if (item instanceof Object) {
      newobj[k] = {}
      deepCopy(newobj[k], item)
    }
    // 3.判断这个值是否为简单数据类型
    else {
      newobj[k] = item
    }
  }
}

# 五、正则表达式

用于匹配字符串中字符组合，在 js 中为对象。

可匹配、替换、提取；灵活性、逻辑性、功能性强。

  1. 创建正则表达式

    // 1.使用ReExp对象创建正则表达式
let regexp = new RegExp(/123/)
console.log(regexp)
// 2.利用字面量创建
let rg = /123/

// 检测字符串是否符合正则表达式要求的规范
console.log(rg.test(123))
console.log(rg.test('124'))

  2. 边界符：^ $ 设定开头和结尾

    let rg = /abc/ //只要包含了此字符串
let reg1 = /^abc/ //以此字符串开头
let reg2 = /^abc$/ //精确匹配

  3. 字符类：

    let rg = /[abc]/ // []只要匹配其中一个即可
let rg1 = /^[abc]$/ //三选一
let rg2 = /^[a-z]$/ //26个字母
// 字符组合
let rg3 = /^[a-zA-Z0-9_-]$/
// [^] 则表示取反
let rg4 = /^[^a-zA-Z0-9_-]$/

  4. 量词符：用来设定某个模式出现的次数

    量词 说明
    * 重复 0-n 次
    + 重复 1-n 次
    ？ 0、1 次
    {n} n 次
    {n, } n 次或更多次
    {n, m} n-m 次
    let rg3 = /^[a-zA-Z0-9_-]{6,16}$/

  5. 预定义类：

    预定义类 说明
    \d [0-9]
    \D [^0-9]
    \w [A-Za-z0-9]
    \W [^A-Za-z0-9]
    \s [\t\r\n\v]
    \S [^\t\r\n\v]

或者：|

  1. 替换：replace

    str.replace(/激情/[switch], '**')

    switch（修饰符）: g 全局匹配；m 执行多次匹配；i 忽略大小写；gi 全局匹配+忽略大小写。

  2. 匹配：exex() 数组的 0 项为匹配的文本


# 六、补充

ECMAScript，由 ECMA 国际标准组织制定的一项脚本语言标准规范。

  1. var：函数级作用域；存在变量提升；可声明多次。

  2. let：块级作用域(大括号包起来的)；不存在变量提升，暂时性死区特性；只能声明一次。

  3. const：常量具有块级作用域；声明常量必须赋值；不可更改（复杂数据的元素可更改，但不能重新赋值）。

  4. 经典面试题：

    展开代码 
    var arr = []
for (var i = 0; i < 2; i++) {
  arr[i] = function () {
    console.log(i)
  }
}
arr[0]()
arr[1]()
// 关键点：两个函数执行的i都是全局作用域下的i

var arr = []
for (let i = 0; i < 2; i++) {
  arr[i] = function () {
    console.log(i)
  }
}
arr[0]()
arr[1]()
// 关键点：循环每一次产生块级作用域，两个函数执行的i都是上一级作用域下的i

  5. 解构赋值：从数组或对象中提取值，赋给另外的变量。

    点击展开代码 
    // 1. 基本
let [a, b, c] = [1, 2, 3] // a=1, b=2, c=3
// 2. 可嵌套
let [a, [b], c] = [1, [2], 3] // a=1, b=2, c=3
// 3. 可忽略
let [a, , b] = [1, 2, 3] // a=1, b=3
// 4. 不完全解构
let [a = 1, b] = [] // a=1, b=undefined
// 5. 剩余运算符
let [a, ...b] = [1, 2, 3] // a=1, b=[a, b]
// 6. 解构默认值
let [a = 3, b = a] = [] // a = 3, b = 3
let [a = 3, b = a] = [1] // a = 1, b = 1
let [a = 3, b = a] = [1, 2] // a = 1, b = 2

  6. 箭头函数： () => {}

    函数体只有一句代码，且代码的执行结果就是返回值，可以省略大括号；

    如果形参只有一个，形参外侧的小括号可省略；

    箭头函数不绑定 this，this 为函数上下文位置的 this 指向；

  7. 剩余参数：(...args) 接收未知个参数

  8. 数组扩展方法：

    扩展运算符：可将数组或对象用逗号转为逗号分隔的参数序列；可合并数组；将伪数组转为真数组。

  9. Array 方法：

    Array.from()：将为数组或对象转真数组；

    find()：用于找出符合条件的数组成员，如果查找不到则为 undefine；

    findIndex()：找出第一个符合条件的索引，如果查找不到则为-1；

    includes()：找出第一个符合条件的索引，如果查找不到则为-1；

  10. String 扩展方法： startsWith()：判断是否以某个字符串开头，返回布尔值;

    endsWith()：判断是否以某个字符串结尾，返回布尔值;

    repeat(n)：将原字符串重复 n 次，返回新字符串；

  11. Set 数据结构：类似于数组，成员值是唯一的

    add(value)：添加某个值；

    delete(value)：删除某个值；

    has(value)：判断某个值是否为 Set 成员，返回布尔值；

    clear()：清除所有值。
