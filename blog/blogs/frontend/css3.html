# 二、选择器

# (1) 基础选择器

  1. 标签选择器：以标签命名的选择器

  2. 类选择器(class)（主要 CSS 美化）：.类名；不可以以数字、中划线开头；可以重复；一个标签可以有多个类名，但需要空格隔开。

  3. id 选择器（主要配合 js 写标签）：#id 值；不可以重复；一个标签只能有一个 id 属性值，一个 id 选择器只能选中一个标签。

  4. 通配符选择器：*；开发中使用少；主要去除默认的 margin、padding。

  <div id="sbox">

</div>

  3. 交集选择器：“（无分隔）”

  4. 属性选择器：“a[target="_blank"]”

# (3) 伪类选择器

  1. 结构伪类选择器

    选择器 声明
    E:first-child { } 匹配父元素的第一个子元素，并且是 E
    E:last-child { } 匹配父元素的倒数第一个子元素，并且是 E
    E:nth-child(n) { } 匹配父元素的第 n 个子元素，并且是 E
    E:nth-last-child(n) { } 匹配父元素的倒数第 n 个子元素，并且是 E

    n 可以为：

    功能 公式
    偶数 2n、even
    奇数 2n+1、2n-1、odd
    找到前 5 个 -n+5
    找到从第 5 个往后 n+5
    <div>
  <h1>这是一个标题</h1>
  <p>这是第一个段落。</p>
  <p>这是第二个段落。</p>
  <p>这是第三个段落。</p>
  <p>这是第四个段落。</p>
</div>

    /* 找父级第n个子元素 是p则生效 */
p:nth-child(2) {
  background-color: aqua;
}

    DANGER

    找父级顺序下的子元素，并且是 E 的话才生效。

    不常用的

    :root 选择到根元素

    :empty 选择到既不能有文本内容也没有后代元素的元素

    :only-child 没有兄弟元素的元素

    :first-of-type 所有兄弟元素中同类型的第一个

    :last-of-type 所有兄弟元素中同类型的最后一个

    :nth-of-type(n) 所有兄弟元素中同类型的第 n 个,n 是个数字

    :nth-last-of-type(n) 所有兄弟元素中同类型的倒数第 n 个，n 是数字

    :only-of-type 没有同类型兄弟元素

  2. 否定伪类选择器（不常用）

    :not(选择器) 排除满足小括号中选择器的元素

  3. 动态伪类选择器

    :link 选择地址没有被访问过的超链接元素

    :visited 选择地址被访问过的超链接元素

    :hover 选择鼠标悬停在上面的元素。

    :active 选择鼠标在上面并且按键按下不松手的元素

    :focus 选择获取焦点的元素。

  4. 目标伪类选择器（不常用）

    :target 选择当前锚点指向的元素。

  5. UI 元素伪类选择器（不常用）

    :enabled 选择可以使用的表单控件（没有设置 disabled 属性）

    :disabled 选择不可以使用的表单控件（设置了 disabled 属性）

    :checked 选择到被选中的单选按钮、复选框、下拉选项（option）

  6. 语言伪类选择器（不常用）

    lang() 根据语言选择元素（lang 属性的值）

# (4) 伪元素选择器

  1. 一般用来装饰小图（由 CSS 模拟出的行内标签效果）

    (1) (父元素)::before：在父元素前加一伪标签

    (2) (父元素)::after：在父元素后加一伪标签

    DANGER

  3. ::first-letter 选择元素中第一字母

  4. ::first-line 选择元素中第一行

  5. ::selection 用于设置被鼠标选中的文字的样式

# 三、CSS 三大特点

# (1) 继承性

文字控制属性都可以继承。（a 标签的 color、h 系列标签的 font-size 会失效）

# (2) 层叠性

当样式冲突时，只有选择器优先级相同时，才能通过层叠性判断结果。

# (3) 优先级

  1. 优先级公式： 继承 < 通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < !important

    WARNING

    注意点：

    • !important 写在属性值的后面，分号的前面！
    • !important 不能提升继承的优先级，只要是继承优先级最低！
    • 实际开发中不建议使用!important.

  1. 权重叠加计算（复合选择器）：

    (行内样式个数、id 选择器个数、类选择器个数、标签选择器个数)从左向右依次比较

    !important 不是继承，权重最高

# 四、属性

# (1) 字体样式

  1. font-size：字体大小（默认 16px）；

  2. font-weight：字体粗细（normal 正常、bold 加粗、light 变细/纯数字）；

  3. font-style：字体样式(是否倾斜)（normal 正常、italic/oblique 斜体）；

  4. font-family：字体类型（微软雅黑、宋体、楷体...）；

  5. font-variant：字体修饰符（normal 普通样式、small-caps 小型大写字母）

三类：sans-serif 无衬线字体（黑体、Arial）、serif 衬线字体（宋体、Times New Roman）、monospace 等宽字体（Consolas、fira code）；

简写（复合属性，不可颠倒顺序）：font: style variant weight size family；

# (2) 文本样式

  1. text-indent：文本缩进（2em 可负）；

  2. text-align：内容对齐方式（left、center、right）；

  3. text-decoration：文本修饰（underline 下划线、line-through 删除线、overline 上划线、none 无装饰）；

  4. text-shadow：文本阴影（水平位移 垂直位移 模糊半径 color），模糊半径可省略；

  5. word-spacing：单词间距（可负）；

  6. letter-spacing：文本间距（可负）；

  7. line-height：行高（设置文字父元素高度可实现垂直居中 1.5 倍、50px）（行高 1 倍就是字号大小）（不可和 font 连写）（不可负）；

  8. text-transform：英文大小写转换（uppercase 全为大写、lowercase 全为小写、capitalize 首字母大写）。

# (3) 背景相关

  1. background-color: red：背景颜色（bgc）

  2. background-image:url(‘...’)：背景图片（bgi）

    /* 半透明渐变：透明transparent->rgba() */
background-image: linear-gradient(transparent, rgba(0, 0, 0, 0.6));
/* 第一个参数可设置方向 */
background-image: linear-gradient(to right bottom, #3498db, #2fd49d);

  3. background-repeat：背景平铺（bgr）

    取值 效果
    repeat （默认值）水平、垂直方向平铺
    no-repeat 不平铺
    repeat-x 沿着水平方向平铺
    repeat-y 沿着垂直方向平铺

  4. background-position：背景位置（bgp）

    取值方式一：方位

    • 水平方向：left、center、right
    • 垂直方向：top、center、bottom

    取值方式二：坐标(x, y)：

    • x：水平向右
    • y：垂直向下

    TIP

    背景相关属性连写：background: color image repeat position;（可以打乱顺序）

    背景图片（不重要的，美化作用）、img（重要）

  5. background-size：缩放背景图（bgs）

    数字+px：简单方便，常用

    百分比：相对于自身盒子的宽高比

    contain：背景图等比例缩放 超出宽高就不缩放

    cover：完全覆盖 可能导致显示不全

# (4) 元素显示模式

display

取值 效果
block 转为块级元素
inline-block 转为行内块元素
inline 转为行内元素

  • 块级元素（div、p、h、ul、li、dl、form...）：独占一行；宽度为父级的 100%；添加宽度都生效。

  • 行内元素（a、span、b、u、i、s）：一行显示多个；宽度高度由内容撑开；不可以设置宽高。

  • 行内块元素（input、textarea、button、select、img）：一行可以显示多个；可以设置宽高。

WARNING

标签嵌套：块级的嵌套行内的，p 里面不可以有 div、h、p，a 不可以嵌套 a。

# (5) 盒子模型

组成：外边距（margin）、边框（border）、内边距（padding）、内容（content）

  1. margin，padding

    参数个数 表示的位置
    4 上、右、下、左
    3 上、左右、下
    2 上下、左右
    1 上下左右

    margin: 0 auto; 可让设置了宽的盒子居中

  2. border 实线（solid）、虚线（dashed）、点线（dotted）...

TIP

关于尺寸设置： (1)手动内减；(2)自动内减（box-sizing:border-box）

代码演示 
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
    <style>
      * {
        padding: 0;
        margin: 0;
      }
      div {
        width: 200px;
        height: 200px;
        background-color: aquamarine;
        border: 1px dashed rgb(132, 182, 38);
        padding: 15px;
      }
      .box1 {
        margin: 30px auto;
      }
      .box2 {
        margin: 10px auto 0px;
      }
    </style>
  </head>
  <body>
    <div class="box1">我最帅！！！</div>
    <div class="box2">我最帅！！！</div>
  </body>
</html>

DANGER

注意事项：

  • 垂直布局的块级元素上下的 margin 会合并（取最大值）。

  • 外边距塌陷：互相嵌套的块级元素，子元素的 margin 会作用于父元素。

    可让父元素设置 border-top 或 padding-top 或 overflow: hidden；转换行内块元素；设置浮动。

  • 行内元素用外边距和内边距无法改变标签垂直位置（用行高设置）。

# (6) 浮动

float: left; 左浮动；float: right; 右浮动；clear: both; 清除浮动。

标准流(文档流)（标签默认排列规则）；浮动顶对齐。

WARNING

(1) 浮动脱离标准流，不在占标准流位置。

(2) 浮动元素比标准流高半个级别，可以覆盖标准流中的元素（半脱离）。

(3) 浮动找浮动，浮动元素可以在上一个浮动元素左右浮动。

(4) 可以一行多个，可设置宽高。

(5) Margin: 0 auto、text-align:center 不生效。

清除浮动：

  • 额外标签法

    在最后一个块级元素添加：clear: both

  • 单伪元素清除法

    .clearfix::after {
  content: '';
  display: block;
  clear: both;
  /* 补充代码 （为了兼容性）*/
  height: 0;
  visibility: hidden;
}

  • 双伪元素清除法（还可解决塌陷问题）(推荐)

    /* .clearfix::before 作用：解决外边距塌陷问题 
外边距塌陷：父子标签，都是块级，子级加margin影响父级位置*/
.clearfix::before,
.clearfix::after {
  content: '';
  display: table;
}
.clearfix::after {
  clear: both;
}

  • 给父元素设置 overflow: hidden;。

# (7) 定位

三大布局方式：标准流、浮动、定位

定位方式（position）

取值 定位方式
static 静态定位
relative 相对定位
absolute 绝对定位
fixed 固定定位

设置偏移值（分两个方向，就近原则，如果都有，则以 left、top 为准）

  1. 相对定位(relative)：占有原来位置（没有脱标）；仍然具有原来的显示模式特点；改变位置参照原来的位置。

  2. 绝对定位(absolute 先找已经定位的父级，有父级有定位以其为参照，有父级无定位则以浏览器窗口为参照)：脱标，不占位；改变标签的显示模式，具备行内块特点。

    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
    <style>
      .father {
        position: relative;

        width: 300px;
        height: 300px;
        background-color: rgb(199, 25, 25);
      }
      .son {
        /* 父级相对定位，子级相对定位（子绝父相） */
        position: relative;
        width: 200px;
        height: 200px;
        background-color: rgb(18, 146, 206);
      }
      .sun {
        position: absolute;
        right: 20px;
        bottom: 40px;

        width: 100px;
        height: 100px;
        background-color: rgb(21, 226, 48);
      }
    </style>
  </head>

  <body>
    <div class="father">
      <div class="son">
        <div class="sun"></div>
      </div>
    </div>
  </body>
</html>

    WARNING

    ​ 注：父级相对定位，子级绝对定位（子绝父相）；

    绝对定位查找父级的方式是逐层往上找父级；

    绝对定位的盒子不能实现左右 margin auto 居中（解决方法是 left：50%，再根据盒子大小加个负的 margin-left，垂直类推）！！！

    位移：transform:translate(-50%,-50%); 自己宽高度的一半。

    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
    <style>
      .box {
        position: absolute;

        /* margin: 0 auto; */
        left: 50%;
        /* margin-left: -150px; */
        top: 50%;
        /* margin-top: -150px; */
        transform: translate(-50%, -50%);

        width: 400px;
        height: 300px;
        background-color: rgb(231, 24, 24);
      }
    </style>
  </head>

  <body>
    <div class="box"></div>
  </body>
</html>

  3. border-radius：边框圆角

    常见取值：数字+px、百分比。

    赋值规则：从左上角开始赋值，然后顺时针赋值，没有赋值的看对角。

    做圆：必须是正方形；border-radius: 50%（最大为 50%）；做胶囊按钮：必须是长方形；border-radius 为高度的一半。

    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
    <style>
      .box {
        margin: 20px auto;
        width: 200px;
        height: 200px;
        background-color: blue;

        /* 1个值：4个角相同 */
        /* border-radius: 20px; */

        /* 4个值：按顺时针 */
        /* border-radius: 10px 20px 40px 70px; */

        /* 2、3个值：看对角 */
        border-radius: 20px 50px;
      }
      /* 圆 */
      .box1 {
        margin: 20px auto;
        /* 必须是正方形 */
        width: 200px;
        height: 200px;
        background-color: aquamarine;
        /* border-radius: 100px; */
        border-radius: 50%;
      }
      /* 胶囊 */
      .box2 {
        /* 必须是长方形 */
        margin: 0 auto;
        width: 300px;
        height: 80px;
        background-color: antiquewhite;
        /* border-radius为高度的一半 */
        border-radius: 40px;
      }
    </style>
  </head>
  <body>
    <div class="box"></div>
    <div class="box1"></div>
    <div class="box2"></div>
  </body>
</html>

# (1) 文字溢出显示省略号

  1. 一行显示

    text-overflow: ellipsis; /*超出部分的文字用省略号代替*/
overflow: hidden; /*超出部分隐藏*/
white-space: nowrap; /*强制不换行（一行显示）*/

  2. 多行显示

    overflow: hidden;
text-overflow: ellipsis;
display: -webkit-box; /*弹性伸缩盒子模型显示*/
-webkit-line-clamp: 2; /*限制在一个块元素显示的文本的行数*/
-webkit-box-orient: vertical; /*设置或检索伸缩盒对象的子元素的排列方式*/

  3. 弹性盒子中

    /* 因为弹性盒子的尺寸可以被内容撑开，不换行的盒子可以撑开这个尺寸 */
width: 0;

# (2) 滚动条样式修改

  1. 修改样式

    /* 整个滚动条 */
::-webkit-scrollbar {
  /* 对应纵向滚动条的宽度 */
  width: 10px;
  /* 对应横向滚动条的宽度 */
  height: 10px;
}

/* 滚动条上的滚动滑块 */
::-webkit-scrollbar-thumb {
  background-color: #49b1f5;
  border-radius: 32px;
}

/* 滚动条轨道 */
::-webkit-scrollbar-track {
  background-color: #dbeffd;
  border-radius: 32px;
}

/* 滚动条没有滑块的轨道部分（一般有前三个就可以了） */
::-webkit-scrollbar-track-piece {
  background-color: purple;
}

  2. 隐藏滚动条：

    ::-webkit-scrollbar {
  display: none;
}

# (3) 其他

  1. 精灵图：项目中将多张小图合并成大图，可以减少服务器发送次数，减轻服务器的压力，提高页面加载速度。（一般用 span、b、i、em 行内标签）

    创建一个盒子，设置尺寸和小图尺寸相同；将精灵图设置为背景图；设置背景图位置 background-position。

  2. 不脱标的有：相对定位、Flex 布局。

    会脱标的有：绝对定位、固定定位、浮动。

  3. 去除输入框的默认样式

    去除默认选中框：

    outline: none;

    去除密码框的小眼睛：

    input[type='password']::-ms-clear,
input[type='password']::-ms-reveal {
  display: none;
}

  4. 使用定位 position 导致无法触发底层 click 等事件的解决方法：

    定位盒子的子元素没有点击事件 Pointer-events: none;

    定位盒子的子元素有点击事件 Pointer-events: auto;

  5. 实用快捷键

    ctrl+W：扩选

    ctrl+D：查找相同单词并光标同步

    shift+alt+left/right：扩选、缩选

    alt+鼠标点击、shift+alt+鼠标点击：多行同时编辑

  6. 某个属性要改为 true，直接填属性名。
