# 移动 web

# 一、CSS3

# (1) 字体图标

展示的是图标，本质是字体（简单、字体颜色单一）

优点：灵活性、轻量级、兼容性、使用方便（下载字体包、使用 下载网址 (opens new window)）

  1. 引入：

    <link rel="stylesheet" href="./iconfont.css" />

  2. 调用：

    <span class="iconfont">&#xe6f9;</span>

如果图标库没有，则上传矢量图生成字体图标。（上传 SVG 图标）

# (2) 平面转换

transform，改变盒子的平面形态（位移、旋转、缩放）

  1. 位移：translate(水平移动距离,垂直移动距离)

    单独设置某个方向移动距离：translateX() & translateY()

    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>位移-基本使用</title>
    <style>
      .father {
        position: relative;
        width: 500px;
        height: 300px;
        margin: 100px auto;
        border: 1px solid #000;
      }

      .son {
        position: absolute;
        width: 200px;
        height: 100px;
        background-color: pink;
        transition: all 2s;
      }

      .son:nth-child(1) {
        top: 0;
        left: 0;
      }
      .son:nth-child(2) {
        top: 0;
        right: 0;
      }
      .son:nth-child(3) {
        bottom: 0;
        left: 0;
      }
      .son:nth-child(4) {
        bottom: 0;
        right: 0;
      }
      .father:hover .son {
        top: 50%;
        left: 50%;
        /* transform: translate(50px,100px); */
        /* 百分比：参考自身尺寸计算结果 */
        transform: translate(-50%, -50%);
        /* 一个值默认为水平 */
        /* transform: translateX(50px); */
        /* transform: translateY(50px); */
      }
    </style>
  </head>
  <body>
    <div class="father">
      <div class="son"></div>
      <div class="son"></div>
      <div class="son"></div>
      <div class="son"></div>
    </div>
  </body>
</html>

  2. 旋转：rotate(角度)

    角度单位为 deg；正为顺时针，负为逆时针；需要配合过渡使用；默认为盒子中心点。

    改变转换原点(transform-origin: 原点水平位置，原点垂直位置)，放在盒子里。

    取值：方位名词：left、top、right、bottom、center；像素单位数值；百分比。

    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>旋转效果</title>
    <style>
      .box {
        width: 250px;
        height: 250px;
        background-color: #7090fa;
        transition: all 2s;
      }

      .box:hover {
        /* 顺时针 */
        transform: rotate(360deg);
        /* 逆时针 */
        transform: rotate(-360deg);
      }
    </style>
  </head>
  <body>
    <div class="box"></div>
  </body>
</html>

    平面的多重转换：

    transform: translate() rotate(); /*不可更改顺序（先旋转会改变坐标轴向）；不可单独设置*/
    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>多重转换</title>
    <style>
      .box {
        margin: auto;
        margin-top: 300px;
        width: 800px;
        height: 200px;
        border: 1px solid #000;
      }
      .son {
        width: 200px;
        height: 200px;
        background-color: aquamarine;
        border-radius: 50%;
        border: 1px solid #000;
        transition: all 3s;
      }
      .box:hover .son {
        /* 先旋转会改变坐标轴向 */
        transform: translate(600px) rotate(720deg);
      }
    </style>
  </head>

  <body>
    <div class="box">
      <!-- <img src="./images/tyre1.png" alt=""> -->
      <div class="son">yyds</div>
    </div>
  </body>
</html>

# (3) 空间转换

3D 转换，使用 transform

  1. 立体位移

    transform: translate3d(x, y, z);
transform: translateX();
transform: translateY();
transform: translateZ();

    perspective 属性实现透视效果：视距，近大远小，一般在 800-1200px（给盒子的父级添加）

  2. 立体旋转（绕对应的轴转）

    transform: rotateZ(360deg);
transform: rotateY(180deg);
transform: rotateX(180deg);

    左手定则：握住轴，大拇指指向轴的正方向，手指弯曲方向为旋转正值方向。

    rotate3d(x,y,z,角度度数)用来自定义旋转轴的位置和旋转角度（x、y、z 为 0~1 的数）。

  3. 立体呈现(transform-style:preserve-3d)

    使子元素处于 3d 空间（默认为 flat(2d)）

    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <title>立体呈现</title>
    <style>
      .cube {
        position: relative;
        width: 200px;
        height: 200px;
        margin: 100px auto;
        background-color: pink;
        transition: all 2s;
        transform-style: preserve-3d;
      }

      .cube div {
        width: 200px;
        height: 200px;
        position: absolute;
        top: 0;
        bottom: 0;
      }

      .front {
        background-color: orange;
        transform: translateZ(300px);
      }

      .back {
        background-color: green;
      }
      .cube:hover {
        transform: rotateY(180deg);
        transform: rotate3d(1, 1, 1, 90deg);
      }
    </style>
  </head>
  <body>
    <div class="cube">
      <div class="front">前面</div>
      <div class="back">后面</div>
    </div>
  </body>
</html>

  4. 立体缩放

    transform: scale3d(0.5, 1, 1); /*三轴缩放*/

# (4) 动画

实现多个状态间的变化过程，动画过程可控（重复播放、最终画面、是否停止）。（过渡只能实现 2 个状态间的变化过程）

构成动画的最小单元：帧或动画帧

  1. 定义动画

    方法一：

    @keyframs 动画名称{
    form{}
    to{}
}

    方法二：

    @keyframs 动画名称{
    0%{}
    10%{}
    15%{}
    100%{}
}        /* 百分比指的是动画总时间占比 */

  2. 使用动画

    animation: 动画名称 动画时长;

    复合属性：

    animation: 动画名称 动画时长 速度曲线 延迟时间 重复次数 动画方向 执行完毕时状态;
/* linear匀速 */
animation: change 2s linear;
/* steps()分步动画 */
animation: change 2s steps(3) 2s 3;
/* infinite无限循环 */
/* alternate具有反向执行效果 */
animation: change 2s infinite alternate;
/* backwards（默认值）停留在最初状态 forwards停留在结束状态*/
animation: change 2s forwards;

    注：必须有动画名称和时间；取值不分先后顺序；2 个时间值，第一个为动画时间，第二个为延时时间。

    属性 作用 取值
    animation-name 动画名称
    animation-duration 动画时长
    animation-delay 延迟时间
    animation-fill-mode 动画执行完毕时状态 forwards:最后一帧状态、backwards:第一帧状态
    animation-timing-function 速度曲线 steps(数字)：逐帧动画
    animation-iteration-count 重复次数 infinite 为无限循环
    animation-direction 动画执行方向 alternate 为反向
    animation-play-state 暂停动画 pausedi 为暂停，通常配合 :hover 使用

  3. 制作步骤：

    (1) 准备显示区域（盒子尺寸为精灵小图尺寸，背景图为精灵图）

    (2) 定义动画（改变背景图位置，移动距离为精灵图宽度）

    (3) 使用动画（steps(n)，n 为小图个数，无限循环）

    /* 多组动画 */
animation:{
    动画1,
    动画2,
    ...
}
    代码演示 
    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
    <style>
      .box {
        width: 140px;
        height: 140px;
        border: 1px solid #000;
        background-image: url('../平面转换/images/bg.png');
        animation: change 2s steps(12) infinite, run 2s forwards;
      }

      @keyframes change {
        to {
          background-position: -1680px 0;
        }
      }

      @keyframes run {
        to {
          transform: translateX(500px);
        }
      }
    </style>
  </head>

  <body>
    <div class="box"></div>
  </body>
</html>

# 二、Flex 布局

# (1) 移动端特点

  1. PC 屏幕大，网页固定版心；移动端屏幕小，网页宽度多为 100%。

  2. 谷歌模拟器

  3. 分辨率 （屏幕尺寸是指对角线的长度(英寸)）

    pc 分辨率（1920*1080），缩放 150%（1920/1.5,1080/1.5）

    物理分辨率(出厂设置)、逻辑分辨率(由软件驱动决定)

    代码参考逻辑分辨率：

    iPhone6/7/8：物理分辨率750*1334；逻辑分辨率375*667；比例为2:1

  4. 视口（网页宽度与设备宽度相同）（设计图一般为二倍图）

    二倍图的作用：写代码是 375，放在了 750 的逻辑分辨率中，让图片分辨率高一些，不让图片失真。

# (2) 引入

  1. 百分比布局（也叫流式布局，宽度自适应，高度固定）

  2. 判断浏览器是否兼容 (opens new window)

  3. 浏览器提倡的布局方式；简单、灵活；不脱标。

  4. 组成部分：弹性容器、弹性盒子、主轴、侧轴/交叉轴。

    • 水平排列：默认主轴在水平，弹性盒子沿着主轴排列

    • 侧轴默认是垂直方向

  5. Flex 布局下，不论什么标签设置宽高都会生效，注意不要再 display 了。

# (3) 使用

  1. 设置主轴的对齐方式（justify-content）

    属性值 作用
    flex-start 默认值，起点开始依次排列
    flex-end 终点开始依次排列
    center 沿主轴居中排列
    space-around 弹性盒子沿主轴均匀排列，空白间距均分在弹性盒子两侧
    space-between 弹性盒子沿主轴均匀排列，空白间距均分在相邻盒子之间
    space-evenly 弹性盒子沿主轴均匀排列，弹性盒子与容器之间间距相等

  2. 设置侧轴的对齐方式（align-items）

    单独控制某个弹性盒子（align-self）

    属性值 作用
    flex-start 默认值，起点开始依次排列
    flex-end 终点开始依次排列
    center 沿侧轴居中排列
    stretch 默认值，弹性盒子沿着主轴线被拉伸至铺满容器

    stretch 默认值（前提是弹性盒子没有高），没有宽则由内容撑开。

  3. 伸缩比（flex:数字）

    修改弹性盒子伸缩比：占用父级剩余尺寸的份数。

  4. 改变排列（主轴）方向(flex-direction)

    属性值 作用
    row 行，水平（默认值）
    column * 列，垂直
    row-reverse 行，从右向左
    column-reverse 列，从下向上

    主轴方向变，侧轴也会随之改变。

  5. 弹性盒子多行排列(flex-wrap: wrap)

    调整行对齐方式(align-content) ，取值与 justify-content 基本相同。

代码演示

代码 1

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>主轴对齐方式</title>
    <style>

        margin: 0;

        display: flex;

        /* 居中 */
        /* justify-content: center; */

# 四、移动适配

屏幕宽度不同，网页元素尺寸不同（等比缩放）

  • rem:目前多数企业在用的解决方案

  • vw/vh：未来的解决方案

# (1) rem

  1. 相对单位；相当于html 标签(根)的字号计算结果；1rem=1html 字号大小。

  2. 媒体查询：检测视口宽度，编写差异化的 CSS。

    @media (width: 320px) {
  html {
    font-size: 32px;
  }
}

  3. 设置 html 标签字号：一般为视口宽度的 1/10。

  4. flexible.js ：可自动进行媒体查询并设置 html 标签字号。

  5. 设置时应为 当前像素/当前根字号。

# (2) vw / vh

与 rem 不同之处：不需要进行媒体查询。

  1. 相对单位；相对视口的尺寸计算结果; 1vw=1/100 视口宽，1vh=1/100 视口高。

  2. 设置时为 当前像素/(1/100 当前视口宽)vw。（vw、vh 只用其一，不混合使用）

# (3) less

快速编译生成 CSS 代码，可以计算、嵌套...

注：浏览器不识别 less 代码，网页要引入对应的 CSS 文件。（EasyLESS 插件保存自动生成 CSS 文件）

  1. 注释：单行// (ctrl+/) 多行/* */(shift+alt+A)

  2. 计算：less 4.0 版本 除法需要加小括号或 ./

  3. 嵌套：可避免写错。

    .父级选择器 {
  //父级样式
  .子集选择器 {
    //子集样式
  }
}

    注：&不生成后代选择器，表示当前选择器，通常配合 hover、伪类或伪元素使用。

  4. 变量：存储数据，方便使用和修改。

    (1) 定义变量：@变量名：值;

    (2) 使用变量：CSS 属性：@变量名;

  5. 导入：@import '文件路径'; ，导入 less 文件，一般加在 less 开头。

  6. 导出：导出 CSS 文件到相应位置。

    (1) 配置 EasyLess 插件，实现所有 less 有相同的导出路径。

    设置->在 setting.json 中设置->在用户设置中添加下面的：

    "less.compile": {
 "out": "../css/"
}

    WARNING

    注：用户设置是对于所有工程，工作区设置是对当前文件项目进行设置的，会自动生成一个.vscode 文件，它可以覆盖用户设置。

    (2) 在 less 文件第一行加 // out: ../css/（还可以取名字name.css）

  7. 禁止导出：// out:false

# 五、响应式

屏幕大小不同，网页布局效果不同（同一套代码适配不同屏幕尺寸）。

# (1) 媒体查询

根据设备宽度的变化，设置差异化的样式。

@media (媒体特性){  /*min-width、max-width*/
    选择器{
        样式
    }
}
/*注css属性都有层叠性*/
/*min-width（从小到大写）*/
/*max-width（从大到小写）*/

例子：

/* 如果检测到视口宽度小于768px，认为为移动端，left隐藏 */
@media (max-width: 768px) {
  .left {
    display: none;
  }
}

完整写法（了解）：

@media 关键词 媒体类型 and (媒体特性){
    css代码
}  /*and、only、not*/

外链样式：

<link rel="stylesheet" media="逻辑符 媒体类型 and (媒体特性)" href="XX.css" />
<!--注：没有@、有括号-->

BootStrap 框架快速开发响应式网页；由推特开发出来的前端 ui 框架，编写了大量 css 样式，允许开发者结合 html 结构及 JavaScript，快速编写网页。

  1. 下载：中文官网 (opens new window)

    row 类自带间距-15px。

    container-fluid 实现宽度 100%。

    注：弹性盒子也有左右自带 15px 的 padding。
