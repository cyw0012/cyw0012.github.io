# HTML5

# 一、基础知识

  1. 网页组成：代码通过浏览器转换为网页。

  2. 浏览器：IE、火狐、谷歌、Safar、Opera。

    不同浏览器渲染引擎不同，解析效果存在差异。

  3. Web 标准：结构（HTML）、表现（CSS）、行为（JavaScript）。

  4. HTML 的注释为 <!--...-->。

# 二、基本标签

  1. h1~h6：标题标签；p 标签：文本标签；br：换行标签；hr：分割线。

    <h1>你好啊！</h1>
<h2>你好啊！</h2>
<h3>你好啊！</h3>
<h4>你好啊！</h4>

<br />
<p>nh</p>

  2. div 盒子标签（css 中还会再介绍）

    <div>我是盒子</div>

  3. 文本标签：不重要的用单个字母，重要的用单词。

    效果 标签（字母） 标签（单词）
    加粗 b strong
    下划线 u ins
    倾斜 i em
    删除线 s del

  4. 图片标签

    <img src="" alt="" />
<!-- alt 属性是图片未加载出来显示的文字;  title属性：鼠标悬停时提示(其他标签也可添加此属性) -->

    绝对路径（工作中一般不用）：盘符开始、网址。

    相对路径（常用）：从当前文件夹找目标文件。

    • 同级目录：”XXX.jpg” 、 “./XXX.jpg”

    • 下级目录：”./XXX(文件夹)/XXX.jpg”

    • 上级目录：”../XXX(文件夹)/XXX.jpg” (../跳出当前文件夹)

    <img src="num1.jpg" alt="" width="200px" />
<img src="./num1.jpg" alt="" width="200px" />
<img src="../../6.19/add/src/assets/images/img1.jpg" alt="" />

  5. 音频标签（<audio> 支持 MP3、Wav、Ogg）：

    相关属性：src 音频路径； controls 显示播放控件；autoplay 自动播放（部分浏览器不支持）；loop 循环播放。

    <audio
  src="../../../CloudMusic/徐梓淳 - 爱丫爱丫.mp3"
  controls
  loop
></audio>

  6. 视频标签（<video> 支持 MP4、WebM、Ogg）：属性与音频属性一样。

    注：视频可自动播，但要静音（muted）！

# 四、SVG

SVG 指可伸缩矢量图形 (Scalable Vector Graphics)

SVG 用来定义用于网络的基于矢量的图形

SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失

SVG 使用 XML 格式定义图形

  1. 体验 SVG

    <svg>
  <circle
    cx="100"
    cy="50"
    r="40"
    stroke="black"
    stroke-width="2"
    fill="red"
  />
</svg>

  2. 矩形（rect）

    x 属性定义矩形的左侧位置

    y 属性定义矩形的顶端位置

    fill 属性定义填充颜色

    fill-opacity 属性定义填充颜色透明度（合法的范围是：0 - 1）

    stroke 属性定义描边颜色

    stroke-width 属性定义描边粗细

    stroke-opacity 属性定义轮廓颜色的透明度（合法的范围是：0 - 1）

    opacity 属性用于定义了元素的透明值 (范围: 0 到 1)。

    <svg>
  <rect
    x="100"
    y="100"
    width="300"
    height="200"
    style="
     fill: antiquewhite;
     stroke-width: 4;
     stroke: aqua;
     fill-opacity: 0.1;
     stroke-opacity: 0.8;
   "
  />
</svg>

<!-- 创建一个圆角矩形 -->

  6. 多边形（polygon）

    points 属性定义多边形每个角的 x 和 y 坐标

    fill-rule：图形填充规则

    • nonzero：判断一个点是否在图形内，从该点作任意方向的一条射线，然后检测射线与图形路径的交点情况。从 0 开始计数，路径从左向右穿过射线则计数加 1，从右向左穿过射线则计数减 1。得出计数结果后，如果结果是 0，则认为点在图形外部，否则认为在内部。

    • evenodd：判断一个点是否在图形内，从该点作任意方向的一条射线，然后检测射线与图形路径的交点的数量。如果结果是奇数则认为点在内部，是偶数则认为点在外部。

  />

# 五、网页结构

  1. 骨架结构标签

    • <!DOCTYPE html> 文档类型声明，告诉浏览器该网页的 HTML 版本。

    • <html lang="en"> 标识网页使用的语言；作用:搜索引擎归类+浏览器翻译；常见语言:zh-CN 简体中文/en 英文。

    • <meta charset="UTF-8"> 规定网页的字符编码（UTF-8 万国码）。

    • <meta http-equiv="X-UA-Compatible" content="ie=edge"> 解决 IE 兼容性。

    • <meta name="viewport" content="width=device-width, initial-scale=1.0"> 宽度=设备宽度：移动端网页的时候要用。

  2. SEO 三大标签（搜索引擎优化）

    让网站在搜索引擎上的排名靠前（竞价排名；让网页制成 html 后缀；标签语义化，在合适的地方使用合适的标签）。

    （1）<title>XXX</title>：网页标题标签。

    （2）<meta name="description" content="XXX">：网页描述标签。

    （3）<meta name="keywords" content="XXX">：网页关键词标签。

  3. ico 图标设置（一般放在根目录）：<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">。
